| 분야 | 세부 업무 |
| --- | --- |
| Unity 기구학 Core 개발 | Line·Plane·Joint 등 기구학 기반 핵심 클래스 설계 및 구현 |
| Unity 디지털 트윈 UI | 재고 현황 대시보드 UI 설계 및 구현, 그리드 셀 상태 색상 표시 및 클릭 팝업 구현, 3D 씬과 UI 패널 레이아웃 설계 |
| Unity 디지털 트윈 동기화 | 센서 값 기반 컨베이어·4-way 셔틀 등 시스템 내 모든 동작 요소의 실시간 움직임 Unity 구현 |
| 그리드 랙 조립·공간 효율 분석 | 그리드 랙 설치를 위한 지그 설계(프레임 정렬·수평·셀 간격 유지) 및 조립 공정 총괄, 평면 보관 대비 그리드 랙 방식의 공간 절약 효과 비교·분석 (단위 면적당 보관 용량, 확장 구성별 효율) |
